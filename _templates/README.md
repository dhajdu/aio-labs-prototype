# Page Templates

Three starter templates — one per layout pattern. Copy the one you need, rename it, and fill in the PLACEHOLDERs.

| File | Pattern | Use for |
|---|---|---|
| `template-hub.html` | A — Hub | Section index pages: `micro-sessions/index.html`, `blog/index.html` |
| `template-toplevel.html` | B — Top-Level | Full-width pages at root: `certification.html`, `coaching.html` |
| `template-subpage.html` | C — Sub-page | Detail pages inside a section: `micro-sessions/ai-browsers.html`, `blog/posts/*.html` |

Each template has a checklist comment at the top listing everything to update before publishing.

See `AIO/Website HTML Style Guide.md` for the full reference.
